Gismo
    GismoはHomepage作成の際にあると便利なHTML Color Coordinating Utilityです。

インストール


    適当なディレクトリに解凍してください。
    GismoはVX-REXXで作成されていますので、実行にはVX-REXXのRuntime Library `VROBJ.DLL'が必要となります。 `VROBJ.DLL'は下記をクリックするかまたはBBSやOS/2の雑誌等から入手することができます。

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)

使用方法

    使用方法はとても簡単です。 まず、Color Listから色の名称を選択するか、RGBの各Sliderを使用して必要な色を作成します。 `Copy' buttonを押すと選択した色の名称またはRGB値（`#'が先頭につきます）がクリップボードにコピーされます。 コピーされたデータを使用しているEditor上にPasteします。

    1. 使用するColor Groupを下記から選択します。初期値は`Color Name'です。 各Color Groupに関しては下記Homepageを参照してください。

      1. Color Name
        • Color values
      2. 216 Color Palette (6x6x6)
        • Netscape Navigator's color palette on Windows

    2. クリップボードにコピーされる値を色の名称にするかRGB値かは`RGB/Name' radio buttonで設定します。

    3. RGB Sliderを使用して色を作成した場合は`Name'が設定されていてもRGB値がクリップボードにコピーされます。

    4. RGB各色をRGB Slider下のSpin Buttonで細かな調整をすることができます。

    5. `Always Copy'機能がEnable時には色が選択される度にデータがクリップボードにコピーされます。

    6. `Fore'と`Back' radio buttonを切り替える事により文字の前景色・背景色の組み合わせを試す事ができます。

    7. `Tag Template'で色指定を含んだ定型TAGを指定することができます。

      • `%c'で色の名前/RGB値を指定します。
        現在選択されている色が darkseagreen の場合...

          Tag TemplateCopied Tag
          %c darkseagreen
          color="%c" color="darkseagreen"

      • `%f'でForeground, `%b'でBackgroundで選択された色の名前/RGB値を指定します。
        現在選択されている色が Foreground : royalblue / Background : mediumvioletred の場合...

          Tag TemplateCopied Tag
          %f royalblue
          %b mediumvioletred
          forecolor="%f" backcolor="%b" forecolor="royalblue" backcolor="mediumvioletred"

      • [登録の方法]
        1. `New' Buttonを押す。
        2. Tag Areaに新規Tagを記述します(下の欄にコピーされるTAGが表示されます)。
        3. `Add' Buttonを押す。

      • [削除の方法]
        1. 削除したいTag Templateを選択する。
        2. `Delete' Buttonを押す。

      • [Movable/Fixedの設定]
          Tag Template WindowをMovableにするには`Movable' check boxをチェックします。

    8. Color Listの順番をName/Hue/Saturation/ValueでSortすることができます。

    9. Color Grid Tableについて

      • Color Grid Table機能を利用することにより、色の選択がより便利になります。
      • `G' Buttonを押すと現在選択されている色を基準にしたGradation TableがColor Grid Table上に表示されます。

    10. Color NumberをHexadecimal(16進数)とDecimal(10進数)で表示することができます。

      • `Hex/Dec Changeover Button' check boxをチェックするとHex/Dec切り替えButtonが表示されます。

    11. `RGB Slider Color' radio buttonでColor SliderのRibbonの色の変化(Dynamic/Fixed)を設定することができます。
